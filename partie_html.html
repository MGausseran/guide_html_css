HTML CSS

Les bases

Le HTML permet de représenter les pages web dans un navigateur, de structurer sémantiquement le contenu et d’intégrer des liens hypertexte. Il est utilisé conjointement avec JavaScript ou PHP, et est représenté dans des feuilles de style, appelées CSS.

Par où commencer ?

Le HTML est un langage de balisage.

Par là, il faut comprendre qu'il est structuré en "poupée russe", chaque balise marquant un passage de niveau.

Les balises HTML sont délimitées par des chevrons ouvrants < et fermants >.
