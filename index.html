Bienvenue !

Coder en HTML/CSS peut être intimidant au premier abord. Alors, pour vous faciliter le travail et vous donner quelques clés, j'ai compilé ici toutes les notes prises au cours de mes formations.

Mais d'abord, quelques explications pour bien comprendre à quoi servent ces deux langages !

Le HTML, qu'est-ce que c'est ?

Le HTML (pour HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web. Il utilise des éléments, appelés balises, pour définir différentes parties d'une page web comme les paragraphes, les titres, les liens, les images, et autres éléments multimédias.

Et le CSS dans tout ça ?

Le CSS est un autre langage qui s'utilise avec le HTML et permet de contrôler la mise en forme et la disposition des éléments sur une page web. Il permet de changer les couleurs, les polices, les espacements, la taille des éléments, les bordures, les ombres, la disposition, et bien plus encore. Sans lui, les sites internet ressembleraient tous à des pages tapées à la machine à écrire.

Maintenant, apprenons-en plus sur ces deux langages !
